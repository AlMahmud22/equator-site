# PhishGuard Downloads

Place your PhishGuard distribution files here:

## Expected Files

- `phishguard-setup.exe` - Windows installer
- `phishguard.dmg` - macOS disk image
- `phishguard.AppImage` - Linux AppImage

## File Naming

Make sure the filenames match exactly as expected by the application:

- Windows: `phishguard-setup.exe`
- macOS: `phishguard.dmg`
- Linux: `phishguard.AppImage`

## Notes

- Files must be placed directly in this directory
- Large binary files should not be committed to Git
- Consider using Git LFS or hosting files externally for production
