# Gecko Chatbot Downloads

Place your Gecko Chatbot distribution files here:

## Expected Files

- `gecko-chatbot-setup.exe` - Windows installer
- `gecko-chatbot.dmg` - macOS disk image
- `gecko-chatbot.AppImage` - Linux AppImage

## File Naming

Make sure the filenames match exactly as expected by the application:

- Windows: `gecko-chatbot-setup.exe`
- macOS: `gecko-chatbot.dmg`
- Linux: `gecko-chatbot.AppImage`

## Notes

- Files must be placed directly in this directory
- Large binary files should not be committed to Git
- Consider using Git LFS or hosting files externally for production
